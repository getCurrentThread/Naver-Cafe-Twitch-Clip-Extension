Naver-Cafe-Twitch-Clip-Extension v0.1.3 (https://github.com/getCurrentThread/Naver-Cafe-Twitch-Clip-Extension/)
  • General
    클립 링크 자동 변환클립 링크 자동 변환 기능을 사용합니다.
  • 동작 방식
    클립 링크 변환 시점 선택 - 페이지 로딩 시: 섬네일을 비디오로 자동 변환
    - 섬네일 클릭 시: 섬네일을 클릭할 때 비디오로 변환
  • 페이지 로딩 시
    페이지 로딩 시점에 변환할 개수 제한 페이지 로딩 시점에 비디오로 변환할 링크의 최대 개수를 설정합니다. 최대 개수를 초과한 클립부터는 링크를 클릭하는 경우에만 비디오로 변환됩니다. 이 개수를 크게 설정하면 클립이 많은 글에서 브라우저가 한참동안 멈출 수 있습니다.
    (Default: 3, Range: 1~200)
  • 첫 번째 클립을 자동 재생페이지 로딩과 동시에 첫 번째 클립을 자동 재생합니다.
  • 클립 자동 재생 시 클립 음소거클립 자동 재생 시 음소거 상태로 시작합니다.
  • 섬네일 클릭 시
    클립 로드 시 클립 자동 재생섬네일 클릭과 동시에 클립을 자동 재생합니다. 이 옵션을 사용하면 클립이 간혹 음소거 상태로 재생될 수 있습니다.
  • 고급 설정
    클립 로드 시 특정 사운드 볼륨(Volume)으로 설정 TIP: Chrome 계열 브라우저는 자동 재생되는 클립을 종종 음소거합니다. 음소거 문제를 피하려면 본 옵션을 사용해보세요(안 될 수도 있음).
  • Volume (Max Volume: 1.0, 음소거: 0.0, 범위: 0.0 ~ 1.0)
  • 비디오 가로 사이즈(%) 클립 가로 사이즈를 결정합니다.
    (Default: 100, Range: 1~100)
  • 비디오 재생 시 다른 재생 중인 비디오 일시정지 Twitch Clip 또는 Naver Video 또는 Youtube 영상을 재생 시, 다른 재생 중인 모든 Naver Video 와 Twitch Clip, Youtube 영상을 일시정지 합니다. 다음 클립을 재생하기 위하여 이전 클립을 정지할 필요가 없습니다. (엄청 편하다!)
  • 네이버 비디오에도 적용 네이버 비디오 관련 재생 문제가 발생 시 본 옵션을 끄세요.
  • 유튜브 영상에도 적용 유튜브 영상 관련 재생 문제가 발생 시 본 옵션을 끄세요.
  • 클립 원본 링크 삭제 클립 링크를 비디오로 변환 시, 본문에 동일한 링크가 존재하는 경우 삭제하여 보기 좋게 만듭니다.
  • 클립 재생 후 추천 영상 숨김 클립 재생 후에 나오는 추천 영상을 숨겨줍니다.
    (추천 영상으로 문제가 있는 영상이 보이는 것을 방지합니다)
  • Twitch Clip 페이지 스타일로 표시 클립 화면을 클릭하여 재생 및 일시정지 되도록 만듭니다. (편하다!)
    일시정지 시 상단 오버레이와 재생 버튼을 숨깁니다. 재생 중 화면을 더블 클릭하여 전체화면을 할 수 있습니다.
  • 편의 기능
    네이버 비디오를 항상 최대 화질로 시작네이버 비디오를 로드할 때 선택 가능한 최대 화질로 자동 설정합니다.
  • 전체화면 스크롤 동작 개선 비디오를 전체화면 한 후 해제했을 때 스크롤이 다른 위치로 변경되는 문제를 개선합니다. 만약 전체화면 시 스크롤과 관련된 문제가 발생한다면 이 기능을 끄십시오.
  • 자주 사용하지 않는 옵션을 숨기기 설정 페이지에서 자주 사용하지 않는 옵션을 숨깁니다.
